# Student Management System

## Overview

This project is a Java-based Student Management System developed as part of my Bachelor of Science in Information Technology studies at North-West University.

The system is designed to manage student information while demonstrating object-oriented programming principles and software development best practices.

- Git
- GitHub
- Visual Studio Code
